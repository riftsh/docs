OpenCursor inherits the full VS Code keybinding set and adds two AI-specific shortcuts. All standard VS Code shortcuts work without any configuration. The table below organises shortcuts by category for quick reference; the full keybinding editor is available via the command palette.

<Note>
  All shortcuts use `Cmd` on macOS and `Ctrl` on Windows and Linux. Where the two platforms differ, both are listed.
</Note>

## AI features

These two shortcuts are unique to OpenCursor and are the primary entry points for AI-assisted coding.

| Shortcut | Action |
|---|---|
| `Cmd+L` / `Ctrl+L` | Open the AI chat panel |
| `Cmd+I` / `Ctrl+I` | Open inline AI code editing at the cursor |

<Tip>
  Use `Cmd+L` to ask broad questions about your codebase, and `Cmd+I` to make targeted edits to the code at your cursor position without switching context.
</Tip>

## Customising keybindings

You can remap any shortcut using the built-in keybinding editor.

<Steps>
  <Step title="Open the keybinding editor">
    Press `Cmd+Shift+P` / `Ctrl+Shift+P` to open the command palette, then search for **Preferences: Open Keyboard Shortcuts**.
  </Step>
  <Step title="Find the command to remap">
    Type the command name or existing shortcut in the search box.
  </Step>
  <Step title="Set a new keybinding">
    Click the pencil icon next to any command and press your desired key combination.
  </Step>
  <Step title="Edit keybindings.json directly">
    Click the **Open Keyboard Shortcuts (JSON)** icon in the top-right of the keybinding editor to edit `keybindings.json` directly for finer control.
  </Step>
</Steps>

## Vim mode keybindings

When Vim mode is enabled, `VimEngine.ts` intercepts key input and provides full modal editing.

<Tabs>
  <Tab title="Enabling Vim mode">
    Open **Settings** (`Cmd+,` / `Ctrl+,`), search for `opencursor.vim.enable`, and set it to `true`. Alternatively, install the `opencursor-vim` extension and reload the window.
  </Tab>
  <Tab title="Normal mode shortcuts">

| Key | Action |
|---|---|
| `h` / `j` / `k` / `l` | Move left / down / up / right |
| `w` / `b` | Word forward / backward |
| `gg` / `G` | Go to first / last line |
| `dd` | Delete line |
| `yy` | Yank (copy) line |
| `p` | Paste |
| `u` | Undo |
| `Ctrl+R` | Redo |
| `/` | Start search |
| `:` | Open command line |

  </Tab>
  <Tab title="Insert mode">

Enter insert mode with `i`, `a`, `o`, or `O`. Press `Escape` to return to normal mode. All standard OpenCursor shortcuts remain available while in insert mode.

  </Tab>
</Tabs>

<Tip>
  If you are migrating from Sublime Text, install the Sublime keybindings extension from the Extensions panel. The `product.json` includes `ms-vscode.sublime-keybindings` as a keymap tip for users with Sublime Text projects.
</Tip>

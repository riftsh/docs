OpenCursor is a free, open-source AI code editor built on VS Code. Unlike subscription-based AI editors, OpenCursor lets you bring your own API keys — connect OpenAI, Anthropic, Gemini, or run fully local models via Ollama with no monthly fees.

## Getting started in 3 steps

<Steps>
  <Step title="Download or build OpenCursor">
    Clone the repository and run `npm install && npm run compile-fast && ./scripts/code.sh`, or download a pre-built release from GitHub.
  </Step>
  <Step title="Add your API key">
    Open **Settings** → **OpenCursor: AI Configuration** and paste your OpenAI, Anthropic, or Gemini API key. For local models, install Ollama and select it as your provider.
  </Step>
  <Step title="Start coding with AI">
    Press `Cmd+L` (macOS) or `Ctrl+L` (Windows/Linux) to open the AI chat panel. Press `Cmd+I` / `Ctrl+I` anywhere in the editor for inline AI editing.
  </Step>
</Steps>
